A sleek, responsive, and keyboard-friendly web calculator built with **vanilla HTML, CSS, and JavaScript**. Designed for simplicity, performance, and modern UI aesthetics.

 **Live Demo:** https://eldorado101.github.io/Calculator/

## Features
-  Dark mode UI with CSS variables & smooth hover/active states
-  Fully responsive layout using CSS Grid
-  Full keyboard support (numbers, operators, `Enter`, `Backspace`, `Esc`)
-  Prevents invalid operator chaining (`++`, `..`, etc.)
- Handles JavaScript floating-point quirks (`0.1 + 0.2 = 0.3`)
-  Safe expression evaluation (no raw `eval()`)
-  Auto-clears on error state
